# Changelog

## 1.0.0

Initial release. `crew` packages an iterative-development loop as four Claude
Code skills — `/crew:flow` (the loop), `/crew:auditor` (correctness review),
`/crew:editor` (written-content review), and `/crew:specialist` (the cross-model
gate to OpenAI Codex) — with a marketplace that sha-pins its `skill-codex`
dependency.
